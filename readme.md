## ZxwyWebSite/NgConf
### 简介
+ 为海纳思系统的Nginx开发的一键配置工具
+ **※不兼容其它系统**
+ 模板参考宝塔面板的配置

### 使用
+ 运行 `./ngconf`
+ 根据提示输入网站信息，即可自动生成站点

### 卸载
+ 所有数据保存在程序目录，使用软链接映射到nginx目录
+ 暂不支持一键卸载，请到站点管理禁用所有站点后删除程序目录

### 其它
+ 暂不支持一键配置SSL，请手动申请证书
+ 部分配置文件需要手动修改
